# Quick Setup Guide

## Step 1: Install Dependencies

```bash
npm install
```

## Step 2: Configure OpenAI API Key

Create a `.env.local` file in the root directory:

```bash
OPENAI_API_KEY=sk-your-actual-api-key-here
```

You can get your API key from: https://platform.openai.com/api-keys

## Step 3: Run Development Server

```bash
npm run dev
```

Visit http://localhost:3000 to see your portfolio!

## Features Included

✅ Modern, responsive design with glassmorphism effects
✅ Smooth scroll navigation
✅ All CV sections (Home, About, Experience, Projects, Education, Achievements, Contact)
✅ AI-powered chatbot (requires OpenAI API key)
✅ Beautiful animations and transitions
✅ Mobile-friendly navigation
✅ Gradient effects and modern UI

## Note

The chatbot will work once you add your OpenAI API key. Without it, the chatbot will show an error message when used.
